# Практична робота №2: JavaScript. Динамічний контент.
У результаті виконання роботи було спроєктовано та реалізовано інтерактивний веб-інтерфейс для моніторингу параметрів Системи безперебійного живлення (ДБЖ).
Під час роботи було вивчено основи мови програмування JavaScript та методи маніпуляції DOM-деревом. Практично закріплено навички роботи з таймерами (setInterval) для автоматичного оновлення даних у режимі реального часу, обробки подій та генерації випадкових показників датчиків. Крім того, було успішно інтегровано бібліотеку Chart.js для візуалізації даних на графіку, реалізовано збереження історії вимірювань за допомогою localStorage та додано функціонал експорту даних у форматі CSV.
